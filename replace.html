Replace

You can add dummy text by adding a range of "text-variables" to the markup. These variables will be replaced with Lorum ipsum text/numbers depending on the variable name.

    
							
  1. {example}
    2.

Replace arguments

  • title
  • excerpt
  • paragraph
  • name
  • phone
  • mail
  • address

Conditional replace

PureFrontend provides functionality for conditional replace. The function will search the hash of the current url for the first argument - before "?" - and output the second argument if matched. You can also provide a third argument after an additional ":" separator to be printed if search fails.

This feature can be handy to add specific classes to elements depending on the pages current state - active for active item in menu etc.

    
							
  1. {example "question-mark" yes:no}
    2.
