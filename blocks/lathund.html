Import & Repeat

  • <!-- import(*) -->
    import templates where (* = path)

  • <!-- repeat(*) -->
    repeat elements where (* = num)

  • <!-- repeat(*:*) -->
    repeat elements a random amount between (* and *) - of times

  • <!-- {i} -->
    write repeat-index (inside repeat loop)
