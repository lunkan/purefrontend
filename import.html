Import

The "import" function will import an external file to the current template and search it's content for executable "PureFrontend" markup. Data will be imported as text so the file ending is not important - you may use .txt, tmpl or what you find handy.

If you are running the files in your local file directory and not on your server - you might get problems with your browser blocking content. Read about workarounds here.

    
							
  1. <!-- import(blocks/header.html) -->
    2.

Repeat

The "repeat" function will clone next element according to the argument provided. The existing element is included so providing "1" as an argument makes no sence.

    
							
  1. <!-- repeat(2) -->
    2.

Random repeat

You can provide 2 arguments separated with ":" in order to random a repeat number between these arguments. The argument below may result in 1-3 additional clones.

    
							
  1. <!-- repeat(2:4) -->
    2.

Repeat index

Any "{i}" inside of a repeated element will be replaced with the curren index of the loop. If the repeated element has nested repeated child element - the "{i}" marker will be replace with the index of the closest parent

    
							
  1. <!-- repeat(2) -->
    2. 
							
  2. <div id="{i}">
    3. 
							
  3.   <p>Hello, I have id {i}</p>
    4. 
							
  4.   <!-- repeat(3) -->
    5. 
							
  5.   <span>And, my id is {i}, </span>
    6. 
							
  6. </div>
    7.

Here is the output from the example above.

    
							
  1. Hello, I have id 1

  2. And, my id is 1, And, my id is 2, And, my id is 3,
    3. 
							
  3. Hello, I have id 2
    4. 
							
  4. And, my id is 1, And, my id is 2, And, my id is 3,
